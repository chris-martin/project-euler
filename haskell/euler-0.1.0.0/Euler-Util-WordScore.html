letterScore :: Char -> Int Source #

Maps the letters ['A'..'Z'] to [1..26], case-insensitively.

>>> map letterScore "AaBZ"
[1,1,2,26]

wordScore :: String -> Integer Source #

The sum of the letter scores in a string.

C   o    l    i   n
3 + 15 + 12 + 9 + 14
>>> wordScore "Colin"
53
